# Network Spoofer

[Baixar](https://cdn.jsdelivr.net/gh/BorisLua/NetworkSpoofer-userscript@main/spoof.user.js)

Teste uma simulação de erro no site, útil para testar a reação do website com erros.

## Instalação

Primeiro, instale o Tampermonkey. (Gerenciador de userscripts muito bom)

Depois, importe o arquivo `spoof.user.js` no Tampermonkey. (Lembre-se de mudar as variáveis e o `@match` com as suas preferências)

Ou, copie todo o conteúdo do link abaixo e cole na aba Console do Dev Tools (só apertar F12):

https://raw.githubusercontent.com/BorisLua/NetworkSpoofer-userscript/main/spoof.user.js
