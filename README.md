# Network Spoofer ![JavaScript](https://img.shields.io/badge/JavaScript-yellow?style=flat-square&logo=javascript&logoColor=black)
Teste uma simulação de erro no site, útil para testar a reação do website com erros.

## Instalação
Primeiro, instale o Tampermonkey. (Gerenciador de userscripts muito bom)

Depois, importe o arquivo `spoof.user.js` no Tampermonkey ou clique no botão de importar ao Tampermonkey. (Lembre-se de mudar as variáveis e o `@match` com as suas preferências)
[![Importar ao Tampermonkey](https://img.shields.io/badge/Importar%20ao%20Tampermonkey-V1-blue?style=for-the-badge&logo=js)](https://cdn.jsdelivr.net/gh/BorisLua/NetworkSpoofer-userscript@main/spoof.user.js)

Ou, copie todo o conteúdo do link abaixo e cole na aba Console do Dev Tools (só apertar F12):

https://raw.githubusercontent.com/BorisLua/NetworkSpoofer-userscript/main/spoof.user.js
